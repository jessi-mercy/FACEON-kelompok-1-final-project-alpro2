Tentang FACEON

Facial Emotion Recognition Online Network

Tentang Proyek

FACEON adalah sistem pengenalan emosi wajah berbasis Convolutional Neural Network (CNN) yang dikembangkan sebagai project akhir mata kuliah Algoritma Pemrograman 2.

Sistem ini mampu mengenali 7 emosi dasar manusia: Marah, Jijik, Takut, Senang, Sedih, Terkejut, dan Netral melalui analisis gambar wajah.

Teknologi

FACEON menggunakan arsitektur deep learning dengan pendekatan:

  • Convolutional Neural Network (CNN) untuk ekstraksi fitur wajah
  • OpenCV untuk preprocessing dan deteksi wajah
  • TensorFlow/Keras untuk implementasi model ML
  • Flask untuk backend web service
  • HTML/CSS/JS untuk frontend responsive
